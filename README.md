# 🌌 Rajshahi Olympiad Society (ROS) — Official Web Platform
![Version](https://img.shields.io/badge/version-1.0.0-00b4d8?style=for-the-badge)
![UI Theme](https://img.shields.io/badge/UI--Theme-Cyber__Dark__Glass-ffd700?style=for-the-badge)
![Tech Stack](https://img.shields.io/badge/Tech--Stack-HTML5__%2F__Tailwind__%2F__JS-4cc9f0?style=for-the-badge)
রাজশাহী অলিম্পিয়াড সোসাইটির (ROS) অফিশিয়াল ওয়েব প্ল্যাটফর্ম। এটি একটি আল্ট্রা-মডার্ন, সাইবার-ডার্ক এবং গ্লাস-মরফিজম (Glassmorphism) থিম ভিত্তিক ওয়েব অ্যাপ্লিকেশন, যা রাজশাহী অঞ্চলের শিক্ষার্থীদের মেধা বিকাশ, অলিম্পিয়াড কার্যক্রম পরিচালনা, সদস্য ব্যবস্থাপনা এবং বিজ্ঞপ্তি প্রকাশের জন্য ডিজাইন করা হয়েছে।
---
## ✨ প্রিমিয়াম ফিচারসমূহ (Key Features)
- **🌌 সাইবার-ডার্ক গ্লাস UI:** সম্পূর্ণ কাস্টম ডিজাইন যা `--dark-bg (#020c1b)` এবং `--card-bg (rgba(10, 25, 47, 0.65))` ব্যবহার করে গভীর মহাকাশ ও হাই-টেক ভাইব তৈরি করে।
- **⚡ গ্লাস শাইন ও ৩D অ্যানিমেশন:** ইন্টারঅ্যাক্টিভ কম্পোনেন্ট ও কার্ডগুলোতে মাউস হোভার করলে সূক্ষ্ম ৩D লিফট এবং নিয়ন গ্লো (`box-shadow`) ইফেক্ট তৈরি হয়।
- **🔍 অ্যাডভান্সড ডাইনামিক ফিল্টারিং:** সদস্য তালিকায় (Member List) কোনো এক্সটার্নাল লাইব্রেরি ছাড়া সম্পূর্ণ পিওর জাভাস্ক্রিপ্ট দিয়ে তৈরি ৬টি কাস্টম সার্চেবল ডার্ক ড্রপডাউন ফিল্টার।
- **📱 ব্রেকিং-প্রুফ রেসপন্সিভনেস:** মোবাইল ডিভাইসে ডেটা টেবিল ভেঙে যাওয়া রোধ করতে ৩টি ফিক্সড কলাম মেকানিজম এবং বাকি ডেটার জন্য ডাইনামিক মডাল পপআপ (Modal Popup) সিস্টেম।
- **🔗 গ্লোবাল কম্পোনেন্ট আর্কিটেকচার:** `navigation.js` ফাইলের মাধ্যমে ডাইনামিক কাস্টম এলিমেন্ট `<ros-header>` এবং `<ros-footer>` রি-ইউজেবল ইন্টিগ্রেশন।

## 🎨 ডিজাইন গাইডলাইন ও ব্র্যান্ড কালার (Design System)
প্রজেক্টের গ্লোবাল ডিজাইন সিস্টেমের মূল কালার কোডসমূহ নিচে দেওয়া হলো:

| কালার টোকেন | হেক্স কোড (Hex) | ব্যবহার ক্ষেত্র |
| :--- | :--- | :--- |
| `Background` | `#020c1b` | মূল গ্লোবাল ব্যাকগ্রাউন্ড (Deep Space Navy) |
| `Card BG` | `rgba(10, 25, 47, 0.65)` | গ্লাস-মরফিজম প্যানেল এবং ফর্ম কার্ড |
| `Panel Inner` | `#040d1a` | ড্রপডাউন এবং টেবিল ইনার ডার্ক সেকশন |
| `Secondary` | `#00b4d8` | নিয়ন স্কাই ব্লু (বাটন, লিংক ও অ্যাক্টিভ স্টেট) |
| `Accent` | `#ffd700` | গোল্ডেন হলুদ (টাইটেল হাইলাইট, ব্যাজ ও নোটিশ) |
| `Text Light` | `#f1f5f9` | প্রধান রিডেবল টেক্সট এবং শিরোনাম |

## 🛠️ টেকনোলজি স্ট্যাক (Technology Stack)
- **ফ্রন্টএন্ড:** HTML5, CSS3, Tailwind CSS (CDN)
- **আইকন ও ফন্ট:** Font Awesome 6.4.0, Google Fonts (Hind Siliguri & Poppins)
- **স্ক্রিপ্টিং:** Vanilla JavaScript (ES6+ Asynchronous Data Handling)
- **অ্যানিমেশন:** AOS (Animate On Scroll), Custom CSS Keyframes
